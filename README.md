# CarbonLens

CarbonLens is an end-to-end carbon credit market intelligence platform built with SQL, Python, and Power BI.

The project analyzes:

- carbon credit issuance
- retirements
- market growth
- project categories
- geographic trends
- integrity risks
- forecasting scenarios

## Tech Stack

- Python
- SQL
- Power BI
- Jupyter
- Pandas
- PostgreSQL
